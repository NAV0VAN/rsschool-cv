Веб-приложение WORK-IN.RU

HTML
Sass
Bootstrap
NodeJS
SQL
Webpack

Учебное веб-приложение WORK-IN.RU разрабатывалось в процессе изучения ряда технологий и построено по клиент-серверной архитектуре на стеке:

  • frontend: Bootstrap + Sass + JavaScript;
  • backend: NodeJS + Express + MySQL;
  • сборка: Webpack.

Использованная методология программирования: функциональное программирование (ООП практически не применялось).

HTML рендерится на сервере (Server-Side Rendering) с использованием шаблонизатора EJS.

Обмен "клиент-сервер" построен на базе фреймворка Express.

Данные хранятся в базе MySQL.

Для геокодирования и визуализации местоположений в приложении используется API карт OpenLayers и API сервиса Dadata.

Деплой приложения выполнен на VPS хостинга Beget (на базе LEMP).

Навыки:

  • адаптивная верстка сайта с использованием Bootstrap, flex, Sass;
  • реализация интерактивности страницы с использованием JavaScript;
  • реализация клиент-серверной архитектуры приложения на базе NodeJS (fetch, promise);
  • сборка дистрибутива для деплоя на сервер с использованием Webpack;
  • работа с Git (репозиторий приложения приватный по понятной причине).
